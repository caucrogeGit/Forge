Bienvenue sur Forge

Framework web artisanal — HTTP/HTTPS natif, routeur, sessions, validation, pool MariaDB et générateurs de code. Sans magie cachée.

Créer une application en 5 étapes

1

Créer la base de données

Configure la connexion MariaDB et crée la base de données.

python cmd/make.py schema:create
2

Initialiser la sécurité

Crée les tables utilisateur, role et le premier compte admin.

python cmd/make.py security:init
3

Générer une entité MVC

Génère en une commande le model, le validator, le controller, les vues HTML et les routes pour une entité. 

python cmd/make.py mvc NomEntite

Exemple : python cmd/make.py mvc Produit

4

Créer la table SQL

Génère et exécute le DDL CREATE TABLE pour l'entité.

python cmd/make.py table:create NomEntite
5

Démarrer le serveur

Lance Forge en mode développement ou production.

python app.py
python app.py --env prod

Autres commandes utiles

Vérifier une entité

make.py check NomEntite

Lister toutes les entités

make.py list

Générer une migration

make.py migration NomEntite

Hasher un mot de passe

make.py security:hash

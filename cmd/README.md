# Dossier `cmd/` historique

Ce dossier contient d'anciens scripts de génération et d'administration issus
des premières itérations de Forge.

Ils ne constituent pas l'interface officielle de Forge 1.0.1. Pour les usages
stables, utiliser la CLI `forge` :

- `forge make:entity`
- `forge make:crud`
- `forge build:model`
- `forge db:init`
- `forge db:apply`
- `forge check:model`

Les scripts `python cmd/make.py ...` sont conservés temporairement comme code
historique et peuvent servir à relire d'anciennes expérimentations, mais ils ne
doivent pas guider la documentation utilisateur ni les nouveaux développements.
